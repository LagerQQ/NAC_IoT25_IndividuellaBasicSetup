#ATmega328P LED Controller

#Embedded systems project written in C for ATmega328P (Arduino Uno) using direct register manipulation.

#Features

#Rotary encoder selects RGB color

#Encoder button enables RGB blink mode

#4 LEDs with independent control

#Button 1 toggles selected LED ON/OFF

#Button 2 resets LED to blink mode

#Potentiometer controls blink speed (ADC)

#Timer interrupt provides millis() timing

#Software debouncing for all buttons

#Hardware

#Arduino Uno (ATmega328P)

#RGB LED

#4 LEDs

#Rotary encoder

#2 push buttons

#Potentiometer

#Key Concepts

#Timer interrupts

#ADC input

#Rotary encoder decoding

#Software debouncing

#Non-blocking embedded design

#Direct register control (DDR, PORT, PIN)
